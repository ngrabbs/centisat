# PV Panel Hardware

## Layout

- `design/`: panel architecture notes and assumptions
- `kicad/`: KiCad source files and libraries
- `manufacturing/`: Gerbers, drill files, and placement outputs
- `releases/`: versioned manufacturing release bundles

## Source of Truth

- Authoring source files are maintained in `kicad/`.
- Manufacturing outputs are maintained in `manufacturing/`.
- Versioned release bundles are maintained in `releases/`.
