# Project Scope and Positioning

## Team Mission

This modular CubeSat senior design project is developed by a team across
multiple subsystems (EPS, flight controller, communications, and payload).

## Baseline Build (Current Senior Design)

The current baseline communications architecture is a discrete amateur satellite
radio chain:

- UHF downlink TX near 437 MHz (70 cm)
- VHF uplink RX near 145.9 MHz (2 m)
- RP2040 control with Si5351A-based frequency generation

This baseline is the primary integration path for the capstone build and test
campaign.

## Author Focus Area

The author of this repository workstream is focused on communications hardware
and RF system validation, aligned with a career direction in satellite
communications engineering and research.

## Exploratory Work (Not Baseline)

Direct-to-cell / NTN modem analysis appears in firmware, analysis, and test
documents as an exploratory payload concept. This track is intentionally
retained for future work but is not required for the baseline senior design
communications milestone.
